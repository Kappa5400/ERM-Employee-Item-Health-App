ERM, enterprise resource management

project: A web app to log employees and company property they are using. Laptop, car, id. Tracks when each item an employee has needs to be updated or exchanged.

goal: track business items and notify when an item needs updating or renewal ect

front end: two tier system, user and boss.
users can view items only, boss can crud stuff.

stack
db: postgres, flyway for migration
back: java, spring
test: junit 5  mockito
front: hmtl, css, react
security: jwt jsonwebtoken
cicd: docker, github actions
deployment: digital ocean
logging: SLF4J
input validation: Jakarta Bean Validation
